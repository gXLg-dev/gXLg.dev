Zum Schutz meiner Webseite vor Bots und Missbrauch verwende ich Cloudflare Turnstile, einen Dienst der Cloudflare Inc.. Cloudflare kann dabei Ihre IP-Adresse und andere technische Daten verarbeiten. Weitere Informationen finden Sie in der Datenschutzerklärung von Cloudflare.
3. Nutzung von Cloudflare Tunnel
Zur Erhöhung der Sicherheit und Gewährleistung eines zuverlässigen Zugriffs wird diese Webseite über Cloudflare Tunnel geleitet, einen Dienst der Cloudflare Inc.. Das bedeutet, dass sämtlicher Datenverkehr zwischen Besuchenden und meinem Server über das Netzwerk von Cloudflare erfolgt. Cloudflare kann dabei technische Daten wie IP-Adressen und Metadaten von Anfragen verarbeiten. Weitere Informationen finden Sie in der Datenschutzerklärung von Cloudflare.
4. Nutzung von Stripe-Zahlungsdiensten
Ich biete Zahlungsabwicklung über Stripe (Stripe Inc.) an. Dabei wird jedoch lediglich ein Link zum Zahlungsportal von Stripe bereitgestellt. Es werden auf meiner Webseite selbst keine Zahlungsdaten verarbeitet. Informationen zur Datenverarbeitung durch Stripe finden Sie in der Datenschutzerklärung von Stripe.

Meine Webseite verwendet Google Fonts, um eine einheitliche und optisch ansprechende Typografie sicherzustellen. Die Schriftarten werden per @import von den Servern von Google geladen. Beim Aufruf meiner Webseite kann Ihr Browser eine Anfrage an Google senden, wodurch Ihre IP-Adresse und andere technische Daten verarbeitet werden können. Weitere Informationen finden Sie in der Datenschutzerklärung von Google.

7. Verwendete Cookies
Ich verwende zwei technische Cookies auf meiner Webseite:
  • token: Wird für die Authentifizierung verwendet. Dieses Cookie speichert eine gehashte Version der IP-Adresse des Nutzers. Es erfolgt keine Weitergabe an Dritte und die Verarbeitung erfolgt ausschließlich auf meinem Server. Der IP-Hash wird nicht auf dem Server gespeichert.
  • state: Wird zur Verwaltung von Fehlermeldungen und Erfolgsnachrichten verwendet.
Diese Cookies sind für die technische Funktionalität der Webseite erforderlich und benötigen gemäß DSGVO keine Einwilligung.
8. Speicherung und Verarbeitung personenbezogener Daten
Ich speichere personenbezogene Daten nur in folgenden Zusammenhängen:

  • In meiner Datenbank wird ein verschlüsselter Hash Ihrer Email-Adresse zur technischen Identifikation gepseichert. Der Hash kann nicht ohne weiteres zurückgerechnet werden und dient ausschließlich zur sicheren Authentifizierung. Es erfolgt keine Weitergabe an Dritte.
